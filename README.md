**Readme de la prueba:**

*Cómo ejecutar:* 
situandose en la raíz del proyecto con "npm run dev" para ejecutar el proyecto en local.

*Despliegue:*
el  proyecto desplegado con Netlify. el enlace del proyecto desplegado: https://pruebajunior.netlify.app

*Librerias, lenguajes y herramientas utilizadas:*

  - React. Me permite crear y utilizar componentes modulares que se actualizan automáticamente cuando se cambia el estado, además es ideal para las aplicaciones SPA. Implementando TypeScript mejora la robustez del código, definiendo tipados para evitar los bugs, proporciona un buen autocompletado y documentación en el editor

  - Vite. Ofrece servidor de desarrollo rápido, los cambios se reflejan instantáneamente en el navegador, soporta TypeScript y React, permite ejecutar el proyecto con "npm run dev" en local para la visualización

  - Tailwind.css. Es un framework CSS que me ha permitido rápidamente aplicar estilos a los elementos con las clases,reduce el tamaño de CSS y es muy personalizable

  - ESLint. Una herramienta que me ha ayudado analizando el código en búsqueda de errores, detecta variables o funciones no utilizadas
